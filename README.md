# programski-jezici
Izvorni kod napravljen za zavrsni ispit iz predmeta Programski jezici

Aplikacija omogućava upravljanje muzičkom kolekcijom kroz rad sa albumima, pesmama, izvođačima i žanrovima.

# **Tehnologije:**

## Backend

Java 24

Spring Boot

Spring Data JPA (Hibernate)

Maven

Baza podataka

MySQL

## Frontend

HTML5

CSS3

Bootstrap 5

Vanilla JavaScript (Fetch API)


# **Funkcionalnosti:**

Prikaz svih pesama

Prikaz detalja pesme

Dodavanje nove pesme

Izmena podataka o pesmi

Brisanje pesme (soft delete)

Dodavanje albuma

Izmena albuma

Brisanje albuma (soft delete)

Upravljanje izvođačima i žanrovima
